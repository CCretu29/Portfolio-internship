Photography blog website

Project Description

This project was created in my third semester. The project was to create a Photography blog website where users can create an account, log in, and buy or download photos. I would be the admin who would upload, edit, and delete photos. The website would have a home page, a gallery page, a blog page, and a contact page. As in semester two, I followed the full process for this project. I created this project using Node.js, MongoDB and testing in Postman.

Technologies Used

  • JavaScript
  • Node.js
  • MongoDB
  • Postman

Project Process

In this section, I will explain the steps I took to complete the project. I did research on my competition, made a project plan, created a timeline and answered research questions. After I gathered all the information that I needed I created a wireframe and tested it on multiple users. With the feedback that I received, I created a high fidelity prototype and tested it as well. I got a lot of useful feedback witch I used to improve my prototype. After my design was validated I started creating the website. I started with the back end, creating the server and the database. I did not created any front end due to time constrains, so I tested all my functionality in Postman. At the end of the project I wrote a project report and an advisory report.
